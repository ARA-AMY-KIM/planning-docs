모달 ① / ② / ③ 경로(URL) · 중복 검증 (서버, 저장 시. 수정 모달은 자기 자신 예외) "이미 사용 중인 URL입니다." 모달 유지 + 입력값 보존

DESCRIPTION

서버 검증 실패(URL·메뉴 코드·메뉴명 중복)와 서버 정합성·네트워크 오류는 실패 토스트(B-2)로 처리 + 모달 유지. 프론트 형식 검증(메뉴 코드 형식)만 포커스 아웃 시 인라인 에러(A-3)로 처리. 필수 미입력은 [저장] 버튼 disabled로만 처리(별도 인라인 에러 미노출). 토스트 전환 시 input border 강조는 적용하지 않음(MVP).
