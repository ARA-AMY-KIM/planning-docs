기획 반영 메모 · 미확정 항목

  • 아이콘/일러스트 디자인 — 자물쇠 SVG 아이콘 + gray 단일 톤 임시 적용. 실제 일러스트 자산 확정 시 교체 필요.
  • C-3 감지 방식 — API 401/403 기준인지, 별도 권한 재조회 로직 기준인지에 따라 실제 구현 범위 달라짐. 개발팀 선확인 필요.
  • 권한 없음 페이지 URL — /access-denied 예시 표기. 실제 라우팅 경로는 개발팀 협의 필요.
  • 원본 URL 보존 여부 — 진입 시 원본 URL을 query param 등으로 보존할지 여부는 기획 결정 필요.
